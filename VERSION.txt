* Minor bug fix for long decimals with trim trailing zeros disabled

1.5.0
---------
* Settings and window positions are now saved on exit (HKEY_CURRENT_USER\Software\VB and VBA Program Settings\Number to Text)

1.4.9

1.4.2

* Added option to generate check-style output

1.2.15

* Minor cosmetic changes

1.2.9
---------
* Added menus
* Added about box
* Added option to automatically copy output
* Added option to trim trailing zeros off the fractional part of numbers before processing (0.10 = one tenth vs. ten hundredths)
* Fixed problem with last letter of last word being trimmed
* Cleaned up interface slightly
